1. Kiedy powstal javascript?
-JavaScript powstał w maju w 1995 roku.
2. Dlaczego javascript jest językiem skryptowym -Język skryptowy – język programowania obsługujący skrypty. Często służący do kontrolowania określonej aplikacji.
-W tym przypadku aplikacją jest przeglądarka.
3. Gdzie jest wykonywany JS?
- JavaScript jest wykonywany przez silnik JavaScriptu w przeglądarce, po tym jak HTML i CSS zostaną skompletowane w stronę internetową
4. Dlaczego JS jest bezpiecznym narzędziem?
- JavaScript został zaprojektowany, aby uniemożliwić atakującym uzyskanie dostępu do Twoich plików lub jakichkolwiek prywatnych informacji. Na przykład ma wsparcie, aby uniemożliwić skryptowi w jednej witrynie przeglądanie danych z innej witryny
5. Czy JS jest językiem obiektowym i dlaczego.
- W JavaScript większość rzeczy jest obiektami, począwszy od najbardziej fundamentalnych rzeczy jak stringi czy tablice, na API przeglądarek zbudowanym na szczycie JavaScriptu kończąc.

menubar=no - Okno nie posiada widocznego paska menu

width=160 - wartość w pixelach; ustawia szerokość okna na 160px
top=200 - wartość w pixelach : ustawia położenie okna względem góry ekranu
left=200 - wartość w pixelach : ustawia położenie okna względem lewej strony ekran
